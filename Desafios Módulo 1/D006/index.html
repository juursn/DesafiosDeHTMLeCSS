O que significa cada abreviação?

  • <a> Direciona para um link
  • <p> Cria um parágrafo
  • <br;> Quebra de linha
  • <ul> Cria uma lista não ordenada
  • <ol> Cria uma lista ordenada
  • <li> Acresentar um item na lista
  • <dt>Para definir um termo brevemente
  • <dd> A descrição da definição do termo
  • <dl> Uma lista de definições
  • <q> Uma citação
